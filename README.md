# Torov Visual

A **Minecraft 1.21.4 (Fabric)** client-side visual mod — a HUD + rendering suite with
animated overlays, world effects, custom shaders and quality-of-life modules.

> Архитектура: ядро **Powder** (модули, события, виджеты, рендер) + портированный визуальный
> слой **Torov Visual** (HUD-элементы, шрифты, фигуры, шейдеры). Всё собирается в **один jar**.

| | |
|---|---|
| **Minecraft** | 1.21.4 |
| **Loader** | Fabric `0.16.10` |
| **Fabric API** | `0.115.1+1.21.4` |
| **Java** | 21 (требуется для сборки) |
| **Artifact** | `torov-visual-1.0-1.21.4.jar` |

## ✨ Возможности

### 🖥️ HUD (`torovvisual.adapter.hud`)
Каждый HUD-элемент — модуль категории **HUD**, виден и переключается в меню. Многие можно
**перетаскивать прямо в открытом чате** (зажать ЛКМ на элементе и тащить).

| Модуль | Описание |
|--------|----------|
| **Watermark** | Логотип «TorovVisual 2.0» + FPS, авто-ширина, ровный текст |
| **DynamicIsland** | Островок по центру сверху: имя, часы, пинг-бары, PVP-таймер, уведомления вкл/выкл модулей |
| **Cooldowns** | Список предметов на кулдауне с полосой прогресса (перетаскиваемый) |
| **TargetHud** | Информация о цели: имя, HP, броня, предмет (перетаскиваемый) |
| **BossBars** | Кастомные босс-бары (скрывают ванильные через миксин) |
| **HotBar / HotKeys / Potions / Armor / PlayerInfo / ScoreBoard / Notifications** | Переоформленные элементы интерфейса |

## 🔧 Сборка

> На машине по умолчанию `java` — это Java 8, которую отвергает fabric-loom. Сборка идёт под **JDK 21**.

```powershell
$env:JAVA_HOME = "C:\Users\mi\.jdks\jdk-21.0.11+10"
.\gradlew build
```

Готовый мод: `build/libs/torov-visual-1.0-1.21.4.jar` — кидается в папку `mods`.
Файл `*-sources.jar` рядом — только исходники, в `mods` не нужен.

## 🧱 Как это устроено

- **Модули** (`powder.client.addon`): каждая функция — `Addon` с типом (HUD / VISUAL / UTILS / …),
  виджетами настроек (`@IWidget`: слайдеры, чекбоксы) и подпиской на события.
- **Событийная система** (`powder.api.event`): `@EventSubscribe` методы; шина обходит всю иерархию
  классов, поэтому HUD-элементы могут подписываться через базовый `HudElement`.
- **Миксины** (`powder.client.mixins`): хуки в рендер игры — `EventRender2D/3D`, скрытие ванильного
  хотбара/босс-бара, замена контура блока, перекраска руки и т.д.
- **Шейдеры** (`assets/mre/shaders/core`): core-программы (json + vsh + fsh) с кастомными юниформами
  (`time`, `screenSize`, `baseColor`, `alpha`) — небо, блок-оверлей.
- **Перетаскивание HUD**: элемент с `movable() = true` регистрирует `Draggable`; `MixinChatScreen`
  позволяет двигать его при открытом чате.
